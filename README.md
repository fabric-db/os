# Fabric OS

Fabric OS is the first-of-its-kind Agentic OS for Fabric DB.

```text
Fabric OS = agentic operating layer.
Juju = framework operator for building Fabric OS.
Operator SDK = operator build framework.
OpenFeature = feature framework.
Penpot = design app as tool.
Kubernetes = deployment substrate.
Operator SDK = operator build framework.
OpenTelemetry = observability standard.
OpenCost = cost observability standard.
CNCF Landscape = framework map.
Every framework = agent surface.
Every app = tool surface.
Platform Agents = product component operators.
Desktop Agent = agentic desktop surface.
Ubuntu Desktop = trusted human workstation layer.
MAAS = Metal as a Service.
Agent = active operator.
Box = living storage agent.
AgentBox = agent and box as one operating unit.
RealBox = digital twin of the box.
OpenBox Protocol = protocol surface for box identity, instruction, record, replay, and federation.
```

```text
Framework Agents turn every framework into an operable agent surface.
Apps become tools.
Fabric becomes the platform.
Platform Agents operate product components.
Ubuntu Desktop gives the human a secure workstation.
Desktop Agent gives the workstation an agentic control surface.
MAAS makes metal programmable.
Agentic OS makes the box active.
AgentBox makes storage operational.
RealBox makes the box observable.
OpenBox makes the box programmable.
```
